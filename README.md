# README

## About

Kansha no Rocket JUMP plugin.

record rocket jump count.

- nj_kansha 1(enable) or 0(disable) default 1
- nj_kansha_tag set your custom tag default "default"

## Install

>https://github.com/withgod/sm-nj_kansha/downloads

tar zxvf nj_kansha.tar.gz -C /path/to/sourcemod
